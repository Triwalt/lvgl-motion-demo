# Logo Sources

These source images are product-site icons used for the LVGL motion demo.

- `claude.ico`: downloaded from `https://claude.ai/favicon.ico`
- `gemini.png`: downloaded from the official `gemini.google.com` HTML icon link, `https://www.gstatic.com/lamda/images/gemini_sparkle_4g_512_lt_f94943af3be039176192d.png`
- `openai.ico`: downloaded from `https://openai.com/favicon.ico`
- `openai_simpleicons_white.svg`: downloaded from Iconify's Simple Icons endpoint for a white OpenAI-style line icon, `https://api.iconify.design/simple-icons:openai.svg?color=%23f4f7fa`
- `openai_simpleicons_white.png`: rendered from `openai_simpleicons_white.svg` for LVGL conversion.
- `deepseek.ico`: downloaded from `https://www.deepseek.com/favicon.ico`

Claude, Gemini, OpenAI favicon, and DeepSeek files are official-domain UI icons. The white GPT/OpenAI line icon is from Simple Icons via Iconify for prototype visual matching. These files are not a full brand asset package or a trademark usage approval.
